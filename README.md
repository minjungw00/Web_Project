# Web_Project

## Info

[Link](minjungw00.com)

개인 포트폴리오 및 블로그를 포스트하는 개인 웹 사이트 서비스

로컬 서버를 직접 구축하면서 웹 서비스 개발의 전반을 학습하고 나아가 실제 서비스를 운영하는 것을 목표로 이 프로젝트를 진행함

## Content

- 개인 포트폴리오
- CS 노트
- 기술 블로그
- 웹 게임(Unity WebGL 별도 구축 예정)
